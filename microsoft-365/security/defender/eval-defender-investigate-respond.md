# <a name="step-6-investigate-and-respond-using-microsoft-365-defender-in-a-pilot-environment"></a>Trin 6. Undersøg og svar ved Microsoft 365 Defender i et pilotmiljø

**Gælder for:**
- Microsoft 365 Defender
