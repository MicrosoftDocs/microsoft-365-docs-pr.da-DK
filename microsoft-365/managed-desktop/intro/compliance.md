# <a name="compliance"></a>Overholdelse af regler og standarder

Når du bruger Microsoft Managed Desktop, giver Microsoft dig et omfattende sæt af overholdelsestilbud. Denne indsats hjælper din organisation med at overholde de forskellige overholdelseskrav.

## <a name="compliance-coverage"></a>Dækning af overholdelse

Microsoft Managed Desktop har opnået følgende certificeringer:

- [ISO 27001 ISMS (Information Security Management Standards)](/compliance/regulatory/offering-ISO-27001)
- [ISO 27701 Privacy Information Management System (PIMS)](/compliance/regulatory/offering-iso-27701)
- [ISO 27017 Kode for fremgangsmåde for kontrolelementer til informationssikkerhed](/compliance/regulatory/offering-ISO-27017)
- [ISO 27018- kodeks for beskyttelse af personlige data i skyen](/compliance/regulatory/offering-ISO-27018)
- [ISO 9001 Quality Management Systems Standards](/compliance/regulatory/offering-ISO-9001)
- [ISO 20000-1 Administration af informationsteknologi](/compliance/regulatory/offering-ISO-20000-1-2011)
- [ISO 22301 Business Continuity Management Standard](/compliance/regulatory/offering-ISO-22301)
- [Cloud Security Alliance (CSA) STAR-udrunding](/compliance/regulatory/offering-CSA-STAR-Attestation)
- [Cloud Security Alliance (CSA) STAR-certificering](/compliance/regulatory/offering-CSA-Star-Certification)
- [Kontrolelementer for tjenesteorganisation (SOC) 1, 2, 3](/compliance/regulatory/offering-SOC)
- [Information Security Registered Assessor Program (IRAP)](/compliance/regulatory/offering-ccsl-irap-australia)
- [Payment Card Industry (FUNKTIONER) Data Security Standard (DSS)](/compliance/regulatory/offering-PCI-DSS)
- [Health Insurance Portability and Accountability Act (HIPAA)](/compliance/regulatory/offering-hipaa-hitech)
- [Health Information Trust Alliance (HITRUST) Common Security Framework (CSF)](/compliance/regulatory/offering-hitrust)

## <a name="auditor-reports-and-compliance-certificates"></a>Revisionsrapporter og overholdelsescertifikater

Du kan finde relevante oplysninger, herunder kontrol og tekniske krav, i [Service Trust Portal (STP)](https://servicetrust.microsoft.com/). Denne portal er det centrale lager til sådanne oplysninger om Microsoft Cloud Service-tilbud. Du kan downloade auditorrapporter, overholdelsescertifikater og meget mere fra [afsnittet Revisionsrapporter](https://servicetrust.microsoft.com/ViewPage/MSComplianceGuide) i STP'en.

> [!NOTE]
> Da Microsoft Managed Desktop kører på Azure, har relevante dokumenter som regel filnavne som "Microsoft Azure, Dynamics 365 og andre onlinetjenester". I disse dokumenter kan du normalt finde Microsoft Managed Desktop under kategorien "Microsoft Online Services" eller "Monitoring + Management".

## <a name="shared-responsibility"></a>Delt ansvar

Overholdelse af skytjenester er et delt ansvar mellem udbydere af skytjenester og deres kunder. Du kan få mere at vide [under Delt ansvar i skyen](/azure/security/fundamentals/shared-responsibility).
