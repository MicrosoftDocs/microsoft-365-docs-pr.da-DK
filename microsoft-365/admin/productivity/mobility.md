# <a name="mobility-insights-score--people-experiences"></a>Score for mobilitetsindsigt – personoplevelser

Produktivitetsscore giver indsigt i din organisations digitale transformationsrejse gennem brugen af Microsoft 365 og de teknologioplevelser, der understøtter den. Din organisations score afspejler målingerne af person- og teknologioplevelsen og kan sammenlignes med benchmarks fra organisationer, der svarer til dine. Mobilitetskategorien er en del af personoplevelsesforanstaltningerne. Du kan få mere at vide ved at se [oversigten over produktivitetsscore](productivity-score.md) og læse [Microsofts erklæring om beskyttelse af personlige oplysninger](https://privacy.microsoft.com/privacystatement).

## <a name="prerequisites"></a>Forudsætninger

Personer i din organisation skal have licens til følgende for at komme i gang med mobilitetsindsigt:

- Microsoft Teams
- Exchange Online
- Word
- Excel
- PowerPoint
- OneNote

Du kan få flere oplysninger under [Tildel licenser til brugere](../manage/assign-licenses-to-users.md).

Når personer har været aktive i ovenstående produkter mindst én gang inden for de sidste 28 dage, begynder du at se indsigten.

## <a name="why-your-organization39s-mobility-score-matters"></a>Derfor er din organisation&#39;mobilitetsscore vigtig

En grundlæggende søjle i organisationens produktivitet er, hvor godt folk kan arbejde fleksibelt, uanset hvor de er. Med Microsoft 365 kan personer bevare forbindelsen med Outlook, Microsoft Teams og Yammer. Andre kan også samarbejde om indhold ved hjælp af Word, Excel, PowerPoint og OneNote fra alle placeringer og platforme.

## <a name="how-we-calculate-the-score"></a>Sådan beregner vi scoren

Vi giver en primær indsigt i oplevelsen, der indeholder de vigtigste målepunkter for denne kategori. Derefter bruges en scorestruktur, der er beskrevet nedenfor, til disse målepunkter til at beregne din organisations score.

### <a name="primary-insight"></a>Primær indsigt

Microsoft 365 gør det muligt at arbejde fleksibelt på tværs af apps, herunder Microsoft Outlook, Word, Excel, PowerPoint, OneNote, Microsoft Teams, Yammer og Skype for Business. Folk kan også arbejde overalt ved at bruge en problemfri oplevelse på tværs af stationære, web- og mobilplatforme. Den primære indsigt ser på de produkter, der er aktiveret for personer i din organisation – og hvor mange af disse personer der er aktive på mindst to platforme.

:::image type="content" source="../../media/primary-mobility.png" alt-text="Diagram, der viser den primære indsigt i mobilitet – personer oplever en produktivitetsscore.":::

1. **Header:** Viser procentdelen af personer i din organisation, der har adgang til en af Microsoft 365 Apps, og som er aktive på mindst ét af disse programmer på mere end én platform.
2. **Kroppen:** Indeholder flere oplysninger om, hvordan brugen af disse programmer på flere platforme kan fremme effektiviteten og tilfredsheden.
3. **Visualisering (aktuel tilstand):** Viser, hvor mange personer der bruger mere end én platform på tværs af stationære computere, mobilenheder og internettet i mindst ét Microsoft 365 (liste nedenfor) på følgende måde:
    - **Vandret søjle** , hvor den blå (farvede) del repræsenterer den procentdel, der er udtrykt i overskriften.
    - **Brøken** på linjen fremhæver den (tæller/nævner), der bruges til at beregne procentdelen i overskriften.
        - Tæller: Antallet af personer i din organisation, der bruger et hvilket som helst program i Microsoft Outlook, Word, Excel, PowerPoint, OneNote og Microsoft Teams, Yammer og Skype på mere end én platform fra stationære computere, internettet og mobilenheder inden for de seneste 28 dage.
        - Nævner: Antallet af personer med licens til Microsoft 365 Apps, Exchange, Yammer, Microsoft Teams eller Skype i mindst 1 af de seneste 28 dage.
    - **Peer benchmark-værdien** for nøglemetrikværdien vises også som en procentdel.
4. **Vis mobilitetsressourcer:** Vælg dette link for at få vist Hjælp-indhold.

### <a name="scoring-framework"></a>Resultatstruktur

Mobilitetsscoren for din organisation måler på organisationsniveau (samlet), om personer bruger Microsoft 365 Apps – Outlook, Teams, Word, Excel, PowerPoint, OneNote, Yammer og Skype  - på tværs af de forskellige platforme – stationær pc, web og mobil.

Scorerne er ikke angivet på individuelt brugerniveau.
