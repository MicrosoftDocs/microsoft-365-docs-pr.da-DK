# <a name="microsoft-365-for-enterprise-overview"></a>Microsoft 365 for Enterprise-oversigt

Microsoft 365 til virksomheder er en komplet, intelligent løsning, der giver alle mulighed for at være kreative og samarbejde sikkert.

Microsoft 365 til virksomheder er udviklet til store organisationer, men den kan også bruges til mellemstore og små virksomheder, der har brug for de mest avancerede sikkerheds- og produktivitetsfunktioner.

## <a name="components"></a>Komponenter

Microsoft 365 for virksomheder består af:

|Tjenester|Beskrivelse|
|---|---|
|Lokale apps og skybaserede apps og produktivitetstjenester|Omfatter både Microsoft 365 Apps for enterprise, de nyeste Office-apps til din pc og Mac (f.eks. Word, Excel, PowerPoint, Outlook og andre) og en komplet pakke med onlinetjenester til mail, fillagring og samarbejde, møder og meget mere.|
|Windows 10 Enterprise|Opfylder både store og mellemstore organisationers behov. Det er den mest produktive og sikre version af Windows til brugerne. For it-fagfolk giver den også en omfattende installation, administration af enheder og apps.|
|Enhedshåndtering og avancerede sikkerhedstjenester|Omfatter Microsoft Intune, som er en skybaseret tjeneste til virksomhedsmobilitetsstyring, som hjælper dig med at være produktiv i forhold til at være produktiv, mens du beskytter dine organisationsdata.|
|||

## <a name="plans"></a>Planer

Microsoft 365 til virksomheder findes i tre planer.

|Plannavn|Funktioner|
|---|---|
|E3|Få adgang til Microsoft 365 kerneprodukter og -funktioner for sikkert at forbedre produktiviteten på arbejdspladsen og fremme innovation.|
|E5|Få adgang Microsoft 365 nyeste produkter og funktioner. Disse omfatter Defender Office 365, sikkerhedsværktøjer og samarbejdsværktøjer. Denne plan indeholder alle E3-funktioner samt avancerede værktøjer til sikkerhed, stemme- og dataanalyse.|
|F3|Forbind med dine medarbejdere på første linje via specialbyggede værktøjer og ressourcer, som de kan bruge til at hjælpe dem med at udføre deres bedste arbejde.|
|||

Hvis du har Microsoft 365 E3, kan du også få disse tilføjelser:

- Identity & Threat Protection
- Overholdelse af & af oplysninger
- [Microsoft 365 E5 Overholdelse](https://www.microsoft.com/microsoft-365/business/e5-compliance)
- Microsoft 365 E5 insider-risiko

Microsoft 365 E3 brugere kan bruge disse tilføjelser til at drage fordel af nogle af de ekstra funktioner, Microsoft 365 E5 indeholder.

Du kan finde flere oplysninger [i Funktioner og egenskaber for hver plan](https://www.microsoft.com/microsoft-365/compare-all-microsoft-365-plans).

## <a name="plan-for-and-deploy"></a>Planlægge og installere

Der er tre måder at planlægge og installere produkter, funktioner og komponenter i Microsoft 365 til virksomheder:

- I partnerskab med FastTrack

   Med FastTrack hjælper Microsoft-teknikere dig med at flytte til skyen i dit eget tempo. Se [FastTrack for Microsoft 365](https://fasttrack.microsoft.com/microsoft365).

- Ved hjælp af Microsoft Consulting Services eller en [Microsoft-partner](https://partner.microsoft.com/)

   Konsulenter kan analysere din nuværende infrastruktur og hjælpe dig med at udvikle en plan for at inkorporere al software og tjenester fra Microsoft 365 til virksomheder.

- Gør det selv

   Start med [netværksoversigten for](networking-roadmap-microsoft-365.md) at opbygge eller bekræfte din eksisterende infrastruktur og dine produktivitetsbelastninger.

Hvis du vil se et eksempel på, hvordan en fiktiv men repræsentativ multinational organisation har implementeret Microsoft 365 til store virksomheder, kan du se [casestudiet om Contoso Corporation](contoso-overview.md).

## <a name="best-together-with-surface-and-the-edge-browser"></a>Bedst sammen med Surface og Edge-browseren

Optimer din brugers integrerede og sikre produktivitet med den bedste kombination af Microsoft 365 til virksomheder, Microsoft Surface-enheder Microsoft Edge browseren. Denne integration på tværs af produkter indeholder:

- En fælles identitets- og logonsikkerhedsinfrastruktur.
- Integrerede lokale apps og skyapps til søgning, samarbejde, produktivitet og overholdelse af regler og standarder.
- Omfattende og integreret sikkerhed til hardware, browser, lokale apps og skyapps.
- En fælles infrastruktur til it-administration af installationer og opdateringer.

Her er et eksempel til en virksomhedsorganisation.
